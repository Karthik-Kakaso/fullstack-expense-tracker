# Full-Stack Personal Finance Dashboard

A responsive, full-stack web application engineered to track daily expenses, manage monthly budgets, and provide interactive financial data visualization.

### 🌐 Live Demo
**[Click Here to View Live App](https://karthikkakaso.pythonanywhere.com)**

---

## 🛠️ Technology Stack

* **Backend:** Python, Flask framework
* **Database:** SQLite (Relational database design, dynamic SQL querying)
* **Frontend:** HTML5, CSS3 (Modern Flexbox UI)
* **Data Visualization:** JavaScript, Chart.js (Dynamic rendering)
* **Deployment & Hosting:** PythonAnywhere (Linux environment)

## ✨ Core Features

* **CRUD Operations:** Securely add, edit, and delete financial records.
* **Smart Budget Tracking:** Real-time calculation of category spending against custom monthly limits, featuring dynamic progress bars and status alerts.
* **Interactive Analytics:** Client-side rendering of financial trends using Chart.js (Doughnut charts for category breakdowns, Stacked Bar charts for monthly trend analysis).
* **Advanced Filtering:** Query the database by date range, specific categories, or custom note searches.
* **Data Portability:** One-click CSV export functionality for offline financial tracking.
* **Responsive Design:** Custom dark-mode UI optimized for both desktop and mobile environments.
